# Part 3 - Backend and Infrastructure

This directory contains code written while studying the third part of a long-form
Go YouTube course.

Part 3 focuses on topics that surround Go backend development: version control,
databases, tooling, deployment-related concepts, and application architecture.
The material shifts from language and runtime mechanics to building and operating
backend systems.
